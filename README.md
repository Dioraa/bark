Bark Pet Care Website

Bark Pet Care is a modern and responsive frontend website designed for a pet care service business.
The website provides a friendly and engaging experience for pet owners by showcasing essential services, helpful dog care tips, and a simple pet shop section.

 Features

- Responsive design (works on mobile, tablet, desktop)
- Hero section with background image
- Dog care tips section
- Pet shop (food, toys, clothing)
-  Services (Grooming, Boarding, Vet care)
-  About section with detailed info
-  Contact section with Google Maps
-  Footer with social media links
